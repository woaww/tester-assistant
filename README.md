## О проекте

Информационная система для **извлечения, генерации, валидации и исправления XPath‑локаторов** элементов веб‑интерфейса с использованием локальной LLM (OpenAI‑compatible API) и автоматизации браузера (Chromium через `browser_use`).

Результаты каждого запуска сохраняются в **централизованное файловое хранилище** `runs/<run_id>/` (JSON/TXT + метаданные + сырые ответы LLM).

Опционально система может сохранять результаты в **PostgreSQL** (каталог проектов/страниц, история запусков, аналитика).

### Режимы работы

- **По тегам**: сбор элементов по выбранным тегам (по сути CSS‑селекторы по тегу).
- **По описанию**: поиск одного элемента по текстовому описанию через `browser_use`.
- **От родительского элемента**: сбор дочерних элементов внутри заданного XPath.
